Detailed Description

data structure defining a single backbone segment

Data structure holding the atom indices of a single backbone segment including dihedral angles. A backbone segment may be incomplete if any of the atoms (C, O, N, or Ca) is missing. Dihedral angles are only valid if the segment is complete.
